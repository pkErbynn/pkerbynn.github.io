## Usage
### Requirements
- Working Laptop
- Text Editor [VS code]
- Web Browser [Chrome, Firefox]
- Git [optional]

### Setup
1 Clone this repository
  ```
  $ git clone https://github.com/pkerbynn/pkerbynn.github.io
  ```
2 Navigate into the directory 
  ```
  $ cd /pkerbynn.github.io.git
  ```
3 Double click to open **index.html** file
